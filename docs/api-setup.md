1. Change directory to `learntocode`
2. Setup the API
   ```
   git clone https://github.com/craigmckeachie/api
   cd api
   npm install
   ```
3. Run the API

   ```
   npm run placeholder
   ```
   Visit http://localhost:3000/
   
    >OR

   ```
   npm run northwind
   ```
   Visit http://localhost:4000/
